# Лілка AI

OpenAI чат для Лілки. Підтримує два режими роботи: чат на екрані з віртуальною клавіатурою та режим для взаємодії через serial інтерфейс.

## Можливості

- Інтеграція з OpenAI API для генерації відповідей. Потрібен API ключ.
- Режим чату з віртуальною клавіатурою на екрані або через serial інтерфейс.
- Налаштування Wi-Fi, OpenAI API ключа та моделі відбувається через веб-інтерфейс.
